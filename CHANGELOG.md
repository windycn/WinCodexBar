# 更新日志

## 0.1.1 - 2026-06-21

### 修复

- 修正 Codex Radar 当前预测字段解析，兼容 `prediction.level`、`probability_24h`、`probability_48h` 与 `expected_window`。
- 修正 Codex 正在运行且尚未关闭时，今日、本周、本月和会话分析无法读取最近用量的问题。
- Token 活动现在会同时读取本地 session jsonl 与 `logs_2.sqlite` 中的 `response.completed` 完成事件，并按会话、时间与 token 结构去重。
- 放宽日志来源过滤，兼容 Codex 将完成事件写入 `codex_otel.*` 或 `codex_core.*` span 文本的情况。

### 优化

- 新安装默认开启随 Windows 启动、保持唤醒与强力唤醒。
- 会话分析会纳入仍在运行的 Codex 会话，不再只依赖已关闭后落盘的 session 文件。
- Release 包补充 SQLite 运行依赖，确保单文件包可以直接读取本机 Codex 日志。

## 0.1.0 - 2026-06-18

首次 Windows 发布版本。

### 新增

- Windows 托盘菜单，支持单击打开快捷菜单、双击打开工作台。
- 多 OpenAI OAuth 账号管理，支持添加、切换、删除、导入和导出。
- 账号额度展示，包含 5 小时 / 7 天额度、订阅类型、健康状态和重置时间。
- 托盘图标使用圆环展示当前账号近 5 小时额度状态。
- 手动模式与聚合模式，可根据使用场景选择账号切换方式。
- 工作台账号视图、Token 活动视图和会话分析视图。
- Token 活动支持今日、本周、本月柱状图和累计热力图。
- 会话分析支持展示活跃/归档会话、最近会话和 Token 最高会话。
- Token 价格预设，可估算美元和人民币成本。
- 保持唤醒与高级防休眠，支持空闲后模拟轻微鼠标移动。
- 设置页支持用量显示方式、Token 单位、自动刷新、健康阈值、模型参数、唤醒策略和开机启动。
- Windows 云母 / Fluent 风格界面、圆角窗口和高分屏布局适配。

### 优化

- 工作台和设置页改为按需加载，减少托盘启动和打开窗口时的卡顿。
- Token 活动和会话分析改为后台扫描，避免 UI 线程阻塞。
- 设置页使用页面缓存，切换栏目时不重复创建控件。
- 刷新用量策略调整为优先刷新当前账号，降低后台刷新频率。

### 注意

- 手动切换账号后，已运行的 Codex 实例通常需要重启或新开实例才能使用新账号。
- 聚合模式会启动本地网关；如果 Codex 是切换前已打开的实例，通常需要重启 Codex 或新开实例后才会接入聚合路由。
- 成本统计为本地估算结果，不等同于官方账单。
